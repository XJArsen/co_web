# webServer

## Function

* 使用Reactor的并发模型的事件处理。
* 使用状态机解析HTTP请求报文，支持解析GET和POST请求
* 访问服务器数据库实现web端用户注册、登录功能，可以请求服务器图片和视频文件
* 实现同步/异步日志系统，记录服务器运行状态

## Environment

* Ubuntu 22
* C++17
* MySql
